# Football Match Data Analysis

## Overview
This project analyzes football match event data to extract performance insights
such as pass accuracy, progressive passes, shots, and fouls using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Work Done
- Cleaned raw football event data
- Calculated pass accuracy and progressive passing metrics
- Generated match-wise performance summaries

## Data Note
Only sample raw data is uploaded. Full datasets are excluded due to size and licensing.
